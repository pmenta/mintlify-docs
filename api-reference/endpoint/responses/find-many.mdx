---
title: "Consultar Respostas"
openapi: "GET /responses"
api: "GET https://requests.stage.judit.io/responses"
description: "Esse endpoint consulta as respostas de buscas realizadas por parâmetros de forma paginada"
---

<ParamField query="page" type="integer" required={false}>
    Define a página dos resultados que você deseja consultar.
</ParamField>

<ParamField query="pageSize" type="integer" required={false}>
    Define o número máximo de resultados que você deseja receber por página;
</ParamField>

<ParamField query="orderField" type="string" required={false}>
    Nome do campo em que se deseja ordenar os buscas;
</ParamField>

<ParamField query="orderDirection" type="string" required={false}>
    Sentido da ordenação: ‘ASC’ ou ‘DESC’;
</ParamField>

<ParamField query="user_id" type="string" required={false}>
    Retorna buscas relacionadas ao usuário;
</ParamField>

<ParamField query="response_type" type="string" required={false}>
    Retorna respostas tipo especificado **"lawsuit"**,**"parties"**, **"attachments"**, **"steps"** ou **"lawsuits"**;
</ParamField>

<ParamField query="request_id" type="string" required={false}>
    Retorna respostas relacionadas a uma busca pelo id;
</ParamField>

<ParamField query="request_status" type="string" required={false}>
    Retorna respostas em que o status da busca podem estar em created, “pending”‘, “started”, “cancelling”, “cancelled” ou “completed”, ou ou mais de um de status [“started”, “completed”];
</ParamField>

<ParamField query="created_at_gte" type="date" required={false}>
    Retorna buscas com a data de criação maior ou igual a informada;
</ParamField>

<ParamField query="created_at_lte" type="date" required={false}>
    Retorna buscas com a data de criação menor ou igual a informada;
</ParamField>
<ParamField query="request_created_at_gte" type="date" required={false}>
    Retorna respostas com a data de criação da busca maior ou igual a informada;
</ParamField>

<ParamField query="request_created_at_lte" type="date" required={false}>
    Retorna respostas com a data de criação da busca menor ou igual a informada
</ParamField>

<RequestExample>
    ```bash Request
    curl -X GET "https://requests.stage.judit.io/responses?page=1&pageSize=10"
    -H "api-key: YOUR_API_KEY"
    ```
</RequestExample>
