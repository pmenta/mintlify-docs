---
title: "Consultar Buscas"
openapi: "GET /requests"
api: "GET https://requests.prod.judit.io/requests"
description: "Esse endpoint consulta as buscas realizadas por parâmetros de forma paginada"
---

<ParamField query="page" type="integer" required={false}>
    Define a página dos resultados que você deseja consultar.
</ParamField>

<ParamField query="pageSize" type="integer" required={false}>
    Define o número máximo de resultados que você deseja receber por página;
</ParamField>

<ParamField query="orderField" type="string" required={false}>
    Nome do campo em que se deseja ordenar os buscas;
</ParamField>

<ParamField query="orderDirection" type="string" required={false}>
    Sentido da ordenação: ‘ASC’ ou ‘DESC’;
</ParamField>

<ParamField query="user_id" type="string" required={false}>
    Retorna buscas relacionadas ao usuário;
</ParamField>

<ParamField query="search_type" type="string" required={false}>
    Retorna buscas do tipo de referência especificado “cpf”, “cnpj”, “oab” ou “lawsuit_cnj”;
</ParamField>

<ParamField query="search_key" type="string" required={false}>
    Retorna buscas relacionadas ao número do CPF, CNPJ, OAB ou processo informado;
</ParamField>

<ParamField query="lawsuit_instance" type="integer" required={false}>
    Retorna buscas onde a instância do processo na busca está de acordo com a informada;
</ParamField>

<ParamField query="origin" type="string" required={false}>
    Retorna buscas cuja origem está de acordo com a informada podendo ser “tracking” ou “api”;
</ParamField>

<ParamField query="origin_id" type="string" required={false}>
    Retorna buscas relacionado ao ID da origem que poder ser um tracking_id(ID de um monitoramento) ou um id gerado pela api;
</ParamField>

<ParamField query="status" type="string" required={false}>
    Retorna buscas cujo status podem ser created, “pending”‘, “started”, “cancelling”, “cancelled” ou “completed”, ou ou mais de um de status [“started”, “completed”];
</ParamField>

<ParamField query="distribution_date_gte" type="date" required={false}>
    Retorna buscas com a data de distribuição maior ou igual a informada;
</ParamField>

<ParamField query="created_at_gte" type="date" required={false}>
    Retorna buscas com a data de criação maior ou igual a informada;
</ParamField>

<ParamField query="created_at_lte" type="date" required={false}>
    Retorna buscas com a data de criação menor ou igual a informada;
</ParamField>
<ParamField query="updated_at_gte" type="date" required={false}>
    Retorna buscas com a data de atualização maior ou igual a informada;
</ParamField>

<ParamField query="updated_at_lte" type="date" required={false}>
    Retorna buscas com a data de atualização menor ou igual a informada;
</ParamField>

<ParamField query="started_at_gte" type="date" required={false}>
    Retorna buscas com a data de execução maior ou igual a informada;
</ParamField>

<ParamField query="started_at_lte" type="date" required={false}>
    Retorna buscas com a data de execução menor ou igual a informada;
</ParamField>

<ParamField query="cancelled_at_gte" type="date" required={false}>
    Retorna buscas com a data de cancelamento maior ou igual a informada;
</ParamField>

<ParamField query="cancelled_at_lte" type="date" required={false}>
    Retorna buscas com a data de cancelamento criação menor ou igual a informada;
</ParamField>

<ParamField query="completed_at_gte" type="date" required={false}>
    Retorna buscas com a data de finalização maior ou igual a informada;
</ParamField>

<ParamField query="completed_at_lte" type="date" required={false}>
    Retorna buscas com a data finalização criação menor ou igual a informada;
</ParamField>
