Bem-vindo ao guia de primeiros passos da JUDIT! Vamos guiá-lo através dos passos necessários para começar a usar nossa API.

## Adquira Seu Plano

Para começar a usar a JUDIT, você precisará adquirir um de nossos planos. Entre em contato conosco para conhecer nossas opções de planos e escolher o que melhor se adapta às suas necessidades.

## Gere Sua API Key

Após a aquisição do plano, você poderá gerar sua API Key através da nossa dashboard. A API Key é necessária para autenticar suas solicitações à nossa API.

1. Acesse a dashboard da <a href="https://app.judit.io" target="blank"> JUDIT </a>.
2. Logue com seu usuário ou crie a sua conta.
3. Vá para a seção "API Keys", nos dados do usuário e clique em configurar.
4. Gere uma nova API Key e salve.
5. Clique novamente no botão configurar da seção "API Keys" e copie a sua API Key.

Lembre-se de manter sua API Key segura. Não a compartilhe publicamente ou com pessoas que não deveriam ter acesso, ela é o seu controle de requisições com a JUDIT.

## Usando a API Key

Com a API Key gerada, agora você pode começar a fazer solicitações para a API da JUDIT. Para isso, você deverá incluir a API Key no parâmetro `api-key` do cabeçalho das suas requisições.

## Buscando Seu Primeiro Processo

Para buscar um processo, você deverá fazer uma requisição POST para a rota `/requests`.

Aqui está um exemplo de como fazer isso usando curl:

```bash
curl -X POST "https://requests.prod.judit.io/requests" \
-H "api-key: YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "search": {
       "search_type": "lawsuit_cnj",
       "search_key": "0010137-92.2023.5.03.0077"
    }
}'
```
